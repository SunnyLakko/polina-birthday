# 📋 Полный Код Проекта — Полина День рождения 18 лет

## 🎯 Описание Проекта

Полноценный React + Vite проект с красивым поздравительным лендингом, разработанный с использованием:
- **React 18** - современный фреймворк
- **Vite** - быстрая сборка
- **Framer Motion 10.16.16** - профессиональные анимации
- **CSS Modules** - стили с корейской эстетикой

---

## 🏗️ Архитектура Компонентов

### 1️⃣ StartScreen.jsx
**Назначение**: Splash-страница с кнопкой для включения фоновой музыки

**Ключевые функции**:
- Плавное появление элементов (fade-in)
- Включение музыки `song.m4a` с громкостью 0.2
- Анимация исчезновения при клике
- Переход к основному контенту

**Props**:
- `onStart: Function` - callback при клике на кнопку

---

### 2️⃣ Hero.jsx
**Назначение**: Первая секция с фото актера и вступительным текстом

**Ключевые функции**:
- Отображение фото с округлыми углами и тенью
- Подзаголовок о использовании AI
- Заголовок "18 лет: Начало твоей эстетики"
- Два параграфа вступительного текста
- Плавное появление при скролле (whileInView)

**Компоненты**:
- ImageContainer с скругленными углами
- TextBlock с двумя параграфами

---

### 3️⃣ MessageSection.jsx
**Назначение**: Основная секция с текстом поздравления и анимированными стикерами

**Ключевые функции**:
- Загрузка текста из text.txt через fetch
- 11 стикеров с бесконечной анимацией левитации
- Стикеры расположены хаотично вокруг текста
- Полупрозрачный контейнер текста с backdrop-filter
- Стикеры видны, но не перекрывают текст на мобильных

**Подкомпонент - Sticker.jsx**:
- Каждый стикер имеет:
  - Случайное расположение (left, top)
  - Случайный поворот (-7.5° до 7.5°)
  - Случайный масштаб (0.6 до 1.1)
  - Бесконечная анимация левитации (3-5 сек)
  - Drop shadow для глубины

---

### 4️⃣ VideoSection.jsx
**Назначение**: Видеоплеер и текст о философии роста

**Ключевые функции**:
- HTML5 видеоплеер с controls
- Видеоформат: congratulations.mp4
- Два блока текста:
  - "Твой путь — твои правила" (основной текст)
  - "Полина, ты — вдохновение" (специальный блок в gradient)
- Специальная подсветка для цитаты "Полина, ты — вдохновение"

---

### 5️⃣ Footer.jsx
**Назначение**: Финальная секция с логотипом и заключительным сообщением

**Ключевые функции**:
- Отображение логотипа
- Пять основных текстовых блоков с разной стилизацией
- Финальное сообщение: "С днем рождения, Полина! ✨"
- Tagline: "Твой день. Твоя история. Твои 18."
- Плавная анимация фона (opacity pulse)

**Текстовые элементы** (разные стили и размеры):
1. CTA - основной текст
2. Wishes - пожелания
3. Message - главный посыл ("Создавай. Люби. Живи на полную мощность")
4. Note - подзаголовок
5. Signature - основной текст поздравления
6. Tagline - финальная фраза

---

### 6️⃣ App.jsx
**Назначение**: Корневой компонент приложения

**Логика**:
- Использует state `hasStarted` для управления отображением
- При `hasStarted === false` показывает StartScreen
- При `hasStarted === true` показывает остальные компоненты
- Callback `handleStart` передается в StartScreen

**Структура разделов**:
```
App
├── StartScreen (если !hasStarted)
└─ Основной контент (если hasStarted)
    ├── Hero
    ├── MessageSection
    ├── VideoSection
    └── Footer
```

## 🎨 Стилевые Решения

### Цветовая Палитра (Korean Aesthetic)
- **Основные цвета**:
  - Бежевый (background): `#fef5f0`, `#faf7fc`
  - Розовый: `#FFB6D9`
  - Лавандовый: `#C99FE3`
  - Фиолетовый-серый: `#9B8FA8`
  - Текст: `#2d2d2d`, `#4a4a4a`

### Эффекты
- **Backdrop Filter**: `blur(20px)` для стеклянного эффекта
- **Box Shadow**: Мягкие тени `rgba(201, 159, 227, 0.2)`
- **Gradients**: Линейные и радиальные градиенты
- **Border Radius**: 25-50px для скругленных углов

### Типография
- **Шрифты**: Roboto, системные шрифты
- **Размеры**: `clamp()` для отзывчивости
  - Заголовки: `clamp(1.8rem, 5vw, 2.8rem)`
  - Текст: `clamp(1rem, 3vw, 1.1rem)`
- **Line Height**: 1.8-1.9 для комфортного чтения
- **Letter Spacing**: 0.3px-1px для минимализма

## ⚡ Анимации (Framer Motion)

### Типы Анимаций

1. **Fade-in & Slide-up** (whileInView):
```javascript
initial={{ opacity: 0, y: 40 }}
animate={{ opacity: 1, y: 0 }}
transition={{ duration: 0.8, ease: 'easeOut' }}
```

2. **Staggered Children**:
```javascript
containerVariants = {
  visible: {
    transition: {
      staggerChildren: 0.2,
      delayChildren: 0.1,
    },
  },
}
```

3. **Levitation (Стикеры)**:
```javascript
y: [0, -10, 0],
transition: {
  duration: 3 + Math.random() * 2,
  repeat: Infinity,
  ease: 'easeInOut',
}
```

4. **Button Interactions**:
```javascript
whileHover={{ scale: 1.05 }}
whileTap={{ scale: 0.95 }}
```

## 📱 Адаптивность

### Breakpoints
- **Mobile**: < 640px (основной фокус)
- **Tablet**: 640px - 768px
- **Desktop**: > 768px

### Mobile Optimizations
- Стикеры становятся менее прозрачными на мобильных (opacity: 0.7)
- Размеры шрифтов уменьшаются с помощью `clamp()`
- Padding уменьшается на мобильных
- Video responsiveness встроена в HTML5 tag
- Text alignment меняется на левое выравнивание на мобильных

## 🔧 Возможные Тонкие Настройки

### Регулировка громкости музыки:
В `StartScreen.jsx` строка:
```javascript
audioRef.current.volume = 0.2 // Измени 0.2 на нужное значение (0-1)
```

### Скорость анимаций:
В каждом компоненте `transition={{ duration: 0.8 }}`
- Увеличь для медленнее
- Уменьши для быстрее

### Цвета:
В CSS модулях - найди нужный hex и измени:
- `#FFB6D9` - розовый
- `#C99FE3` - лавандовый
- `#9B8FA8` - серо-фиолетовый

### Размеры шрифтов:
Используются `clamp()` для адаптивности:
```css
font-size: clamp(1.8rem, 5vw, 2.8rem);
/* min, preferred (% viewport), max */
```
