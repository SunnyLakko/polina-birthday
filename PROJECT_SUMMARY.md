## ✨ Ключевые Компоненты

### 1️⃣ **StartScreen** 
- **Файлы**: `StartScreen.jsx`, `StartScreen.module.css`
- **Функция**: Красивая splash-страница с кнопкой
- **Анимация**: Fade-in элементов с задержками
- **Функционал**: 
  - При клике включает музыку (громкость 0.2)
  - Растворяется и показывает основной контент

### 2️⃣ **Hero**
- **Файлы**: `Hero.jsx`, `Hero.module.css`
- **Функция**: Первая главная секция
- **Элементы**:
  - Фото актера (uk.png) с округлыми углами
  - Подзаголовок о AI
  - Заголовок "18 лет: Начало твоей эстетики"
  - Два параграфа вступительного текста
- **Анимация**: whileInView с staggerChildren

### 3️⃣ **MessageSection**
- **Файлы**: `MessageSection.jsx`, `MessageSection.module.css`, `Sticker.jsx`, `Sticker.module.css`
- **Функция**: Основной блок с текстом и стикерами
- **Функционал**:
  - Загружает текст из text.txt через fetch
  - Отображает 11 анимированных стикеров
  - Стикеры летают вокруг текста с левитирующей анимацией
- **Анимация**: Infinite levitation (y: [0, -10, 0])

### 4️⃣ **VideoSection**
- **Файлы**: `VideoSection.jsx`, `VideoSection.module.css`
- **Функция**: Видеоплеер и текст о личном пути
- **Элементы**:
  - HTML5 видеоплеер с controls
  - Два текстовых блока
  - Специальный акцент "Полина, ты — вдохновение"
- **Анимация**: whileInView

### 5️⃣ **Footer**
- **Файлы**: `Footer.jsx`, `Footer.module.css`
- **Функция**: Финальная секция
- **Элементы**:
  - Логотип
  - 6 текстовых блоков с разными стилями
  - Финальное сообщение и tagline
- **Анимация**: Pulse фона + все элементы

### 🎛️ **App** (Root Component)
- **Файлы**: `App.jsx`, `App.module.css`
- **Логика**: Управляет состоянием `hasStarted`
- **Поток**: StartScreen → основной контент

## 🎨 Дизайн и Стиль

### Палитра Цветов (Korean Aesthetic)
```
Основной цвет: #C99FE3 (лавандовый)
Акцент: #FFB6D9 (розовый)
Нейтральный: #9B8FA8 (серо-фиолетовый)
Текст: #2d2d2d, #4a4a4a
Фон: #fef5f0, #faf7fc (бежевый/сиреневый)
```

### Типографика
- **Шрифт**: Roboto + системные шрифты
- **Заголовки**: `clamp(1.8rem, 5vw, 2.8rem)`
- **Текст**: `clamp(1rem, 3vw, 1.1rem)`
- **Line Height**: 1.8-1.9
- **Letter Spacing**: 0.3px-1px

### Эффекты
- Backdrop filter blur(20px)
- Мягкие тени
- Скругленные углы 25-50px
- Градиентные фоны
- Полупрозрачные контейнеры

## ⚡ Анимации (Framer Motion)

### Типы Анимаций:

1. **Fade-in & Slide-up**
   ```javascript
   initial={{ opacity: 0, y: 40 }}
   whileInView={{ opacity: 1, y: 0 }}
   ```

2. **Staggered Children**
   ```javascript
   staggerChildren: 0.2
   delayChildren: 0.1
   ```

3. **Infinite Levitation** (Стикеры)
   ```javascript
   y: [0, -10, 0]
   repeat: Infinity
   duration: 3-5 сек
   ```

4. **Hover Effects**
   ```javascript
   whileHover={{ scale: 1.05 }}
   whileTap={{ scale: 0.95 }}
   ```

## 📞 Дополнительная Поддержка

### Если что-то не работает:

1. **Музыка не играет** → Проверь `/public/song.m4a` и права доступа
2. **Видео не загружается** → Исправь опечатку, проверь разрешение
3. **Стикеры не видны** → Убедись, что `stick.png-stick11.png` существуют
4. **Текст не загружается** → Проверь `text.txt` в публичной папке
5. **Ошибка сборки** → Очисти `node_modules` и переустанови
