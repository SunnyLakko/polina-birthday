# 🚀 Быстрый Старт

## 📦 Шаг 1: Установка и Подготовка

### Открыть терминал в папке проекта и выполнить:

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev
```

Проект откроется на **http://localhost:3000**

---

## 🎨 Шаг 2: Подготовить Медиа Файлы

### Поместить в папку `/public/`:

1. **song.m4a** - Фоновая музыка
2. **uk.png** - Фото (рекомендуемо 400x500px)
3. **stick.png до stick11.png** - 11 стикеров (PNG с прозрачностью, 100-200px)
4. **congratulations.mp4** - Видео (рекомендуемо 1080x1080px)
5. **logo.png** - Логотип (рекомендуемо 200x200px, PNG с прозрачностью)

> **Совет**: Используй оптимизированные изображения для быстрой загрузки на мобильных!

---

## 📝 Шаг 3: Текст Готов

Текст из `text.txt` **уже загружается автоматически** в компонент MessageSection.

Если нужно отключить загрузку текста и вставить свой текст вручную:

Открой [MessageSection.jsx](src/components/MessageSection.jsx) и замени:

```javascript
{loading ? (
  <p>Загрузка...</p>
) : text ? (
  <div>{/* текст здесь */}</div>
) : null}
```

## 🎯 Шаг 4: Как Это Работает?

### При загрузке сайта:

1. ✨ **Появляется StartScreen** - кнопка "Открыть"

2. 🔘 **Клик на кнопку**:
   - Начинает играть музыка (song.m4a)
   - StartScreen исчезает

3. 📸 **Секция 1**: Фото исчезает с анимацией fade-in

4. 💭 **Секция 2**: Текст + стикеры с левитирующей анимацией

5. 🎬 **Секция 3**: Видеоплеер появляется при скролле

6. 🎉 **Секция 4**: Логотип и финальное сообщение

## 🎨 Как Менять Цвета?

Все цвета находятся в CSS модулях каждого компонента.

### Основная палитра:

```css
/* Розовый -> Лавандовый */
background: linear-gradient(135deg, #FFB6D9 0%, #C99FE3 100%);

/* Бежевый градиент */
background: linear-gradient(135deg, #fef5f0 0%, #faf7fc 100%);

/* Мягкий серо-фиолетовый */
color: #9B8FA8;
```

**Просто найди и замени hex-коды на свои!**

## 🎬 Как Менять Анимации?

Все анимации используют **Framer Motion**.

### Быстрые примеры:

**Сделать медленнее:**
```javascript
transition={{ duration: 1.5 }} // было 0.8
```

**Увеличить эффект движения:**
```javascript
initial={{ opacity: 0, y: 80 }} // было y: 40
```

**Остановить левитацию стикеров:**
```javascript
// В Sticker.jsx закомментируй:
// y: [0, -10, 0],
```

## ⚠️ Решение Проблем

### Музыка не играет?
- Проверь, есть ли `song.m4a` в папке `/public/`
- Попробуй другой браузер (Chrome перепроверяет CORS)
- На мобильном может потребоваться пользовательское действие

### Видео не загружается?
- Убедись, что `congratulations.mp4` в `/public/`
- Проверь размер видео (может быть слишком большой)
- Используй MP4 кодек H.264

### Стикеры не видны?
- Проверь, что файлы `stick.png` до `stick11.png` в `/public/`
- Убедись, что это PNG с прозрачностью (не JPG)

### Текст не загружается?
- Проверь, что `text.txt` в корне проекта
- Убедись, что сервер запущен (`npm run dev`)
- Открой DevTools (F12) → Console и проверь ошибки
