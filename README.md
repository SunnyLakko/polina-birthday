# Полина — Поздравительный Лендинг на 18 лет ✨

Красивый, современный поздравительный лендинг с корейской эстетикой, гладкими анимациями и полной адаптацией под мобильные устройства.

## 🎨 Стиль и Дизайн

- **Korean Aesthetic**: Пастельные тона (нежно-розовый, персиковый, светло-сиреневый)
- **Минимализм**: Скругленные углы, много воздуха, мягкие тени
- **Mobile-First**: Оптимизирован для просмотра со смартфона
- **Плавные анимации**: Используется Framer Motion для высокого качества

## 📱 Структура Сайта

1. **Стартовый экран** - Красивая splash-страница с кнопкой "Открыть"
2. **Секция 1 (Встреча)** - Фото актера Ли Дон Ука с подзаголовком
3. **Секция 2 (Текст и Стикеры)** - Основной текст поздравления со стикерами
4. **Секция 3 (Видео)** - Видеоплеер с дополнительным текстом
5. **Секция 4 (Футер)** - Логотип и финальное сообщение

## 🛠️ Технический Стек

- **React 18** - UI Framework
- **Vite** - Build tool
- **Framer Motion** - Анимации и движение
- **CSS Modules** - Стилизация с scope

## 🚀 Установка и Запуск

### Требования
- Node.js (v16 или выше)
- npm или yarn

### Инструкции

1. **Установить зависимости:**
```bash
npm install
```

2. **Запустить разработчик сервер:**
```bash
npm run dev
```

3. **Собрать для production:**
```bash
npm run build
```

## 🎵 Подготовка Медиафайлов

Все медиафайлы должны быть размещены в папке `public/`:
- `song.m4a` - фоновая музыка (будет играть с громкостью 0.2, зациклено)
- `uk.png` - фото актера Ли Дон Ука
- `stick.png` до `stick11.png` - 11 стикеров для анимации
- `congratulations.mp4` - видео поздравление
- `logo.png` - логотип для футера
- `text.txt` - текст поздравления (загружается динамически)

## ✨ Ключевые Функции

✅ Плавные появления элементов (fade-in, slide-up)  
✅ Бесконечная анимация левитации стикеров  
✅ Фоновая музыка с мягкой громкостью  
✅ Полная адаптация под мобильные устройства  
✅ Использование Intersection Observer для анимации при скролле  
✅ Текст загружается из файла text.txt  
✅ Красивая типография и палитра цветов  

## 🎯 Примечания

- Все компоненты используют React Hooks
- Анимации построены на Framer Motion с использованием whileInView
- Стили использют CSS Modules для избежания конфликтов
- Сайт полностью отзывчив и оптимизирован для всех устройств
- Все фразы взяты из файла text.txt, без генерации AI текстов
