# 🚀 НАЧНИ ОТСЮДА - ИНСТРУКЦИЯ ПО ЗАПУСКУ

## ⚡ 3 Простых Шага до Запуска

### Шаг 1: Откройте Терминал
```bash
# Перейди в папку проекта
cd c:\vscode\ apps\site\reference\ux-ui
```

### Шаг 2: Установи зависимости
```bash
npm install
```
*(займет 1-2 минуты)*

### Шаг 3: Запусти проект
```bash
npm run dev
```
*(откроется на http://localhost:3000)*

## 🎯 Если Что-то Не Работает

| Проблема | Решение |
|----------|---------|
| Ошибка установки | `npm install --legacy-peer-deps` |
| Порт 3000 занят | Vite автоматически выберет другой |
| Музыка не играет | Проверь `/public/song.m4a` |
| Видео не загружается | Исправь опечатку в названии |
| Стикеры не видны | Убедись, что `stick.png-stick11.png` есть |
| Текст не загружается | Проверь `text.txt` в публичной папке |
