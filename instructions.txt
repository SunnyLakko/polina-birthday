Выступи в роли Senior Frontend-разработчика и UX/UI дизайнера. Твоя задача — написать код  для красивого, современного поздравительного лендинга.

Стиль и дизайн:

Korean Aesthetic: Использование пастельных тонов (нежно-розовый, персиковый, светло-сиреневый), скругленные углы, много «воздуха» (отступов), мягкие тени, аккуратный минимализм.

Адаптивность: Сайт создается в первую очередь для девушки (18 лет), которая будет смотреть его со смартфона. Верстка должна быть идеально оптимизирована под мобильные устройства (Mobile-First).

Плавность: Все элементы должны появляться очень плавно (используй Intersection Observer API для fade-in и slide-up анимаций при скролле). Сайт должен ощущаться «живым».

Структура и логика работы (строго по порядку):

Стартовый экран (для обхода автоплея аудио): Красивый стартовый экран с кнопкой (например, "Открыть"). При клике на неё плавно начинает играть фоновая музыка (файл song.m4a, громкость 0.2, зациклено), стартовый экран растворяется и открывается основной контент.

Секция 1 (Встреча): Сразу после клика плавно (с анимацией fade-in и небольшим движением снизу вверх) появляется картинка актера Ли Дон Ука (файл uk.png). Прямо под этим фото расположен аккуратный полупрозрачный подзаголовок: "использован ии в целях производства сайта".

Секция 2 (Текст и Стикеры): Основной блок для слов поздравления. Текст генерировать НЕ НУЖНО, просто сделай JS-функцию, которая через fetch('text.txt') подгружает текст из файла и вставляет его в красивый контейнер (или сделай четкий HTML-плейсхолдер, куда я сам вставлю текст из text.txt).

Стикеры: По всей этой секции, вокруг текста, должны быть красиво и слегка хаотично «развешаны» стикеры (файлы от stick.png до stick11.png). Задай им бесконечную анимацию левитации (плавное покачивание/движение вверх-вниз на пару пикселей), чтобы они казались живыми. При прокрутке с телефона они не должны перекрывать сам текст.

Секция 3 (Видео): По мере скроллинга плавно появляется видеоплеер (файл congrulations.mp4). Под видео снова должен быть подзаголовок: "использован ии в целях производства сайта".

Секция 4 (Футер): В самом конце страницы плавно выплывает логотип (файл logo.png).

Технические требования:

Используй современный стек: React (сборщик Vite).

Для стилизации используй CSS-модули (или Tailwind CSS, если тебе так удобнее) для соблюдения пастельного корейского стиля.

Для плавных появлений и анимации стикеров обязательно используй библиотеку Framer Motion — анимации должны быть высшего качества, очень мягкими и реагирующими на скролл (whileInView).

Разбей проект на логичные React-компоненты (например: Hero.jsx, MessageSection.jsx, VideoSection.jsx, Footer.jsx).

Не пиши тексты поздравлений от себя — используй только те фразы, которые я указал. Текст поздравления подгружай из отдельного файла или создай пропс/константу, куда я смогу его вставить. текст из файла text.txt
